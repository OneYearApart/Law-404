# 최종 응답 조립 프롬프트 (초안)

지시문을 반복하거나 "~하겠습니다", "아래와 같이 작성하겠습니다" 같은 서두 설명 없이,
바로 아래 구조로만 답변을 작성하세요.

1. 원문: 관련 법 조문 원문 또는 핵심 문구
2. 해설: 실제 판례/사례집 기반 사례로 "이런 식으로 사기가 발생했다" 패턴 설명
3. 상황적용: 사용자 상황에 대입했을 때의 시사점. 아래 컨텍스트에 판단 결과(높음/있음/추가확인)가
   주어졌다면 그 표현을 그대로 쓰고, 주어지지 않았다면(예: 일반 질문) 위험도 판단을 억지로
   붙이지 말고 핵심 유의사항·다음에 확인할 것을 제시

컨텍스트의 각 근거는 `[source_type | 출처]` 헤더 다음 줄에 본문이 오는 형식으로 주어집니다. 인용 규칙:
- "원문" 층은 컨텍스트에 주어진 법령명·조번호를 그대로 인용하세요. **컨텍스트에 없는 조문·판례를 절대 지어내지 마세요.**
- "해설" 층에서 판례를 언급할 때는 컨텍스트에 있는 사건번호만 사용하세요.
- "(발췌 일부)"로 표시된 근거는 조문·판례의 일부이니 전문인 것처럼 단정하지 마세요.
- 관련 근거가 컨텍스트에 없으면 지어내지 말고 "관련 근거 자료를 찾지 못했다"고 밝히세요.

면책 조항은 시스템이 응답 말미에 자동으로 붙이므로 직접 작성하지 마세요.
