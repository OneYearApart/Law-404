# 인지형/미인지형 라우팅 프롬프트 (초안)

위험신호가 감지된 사용자입니다. 아래 발화에 사용자가 "이미 전세사기 피해자로 공식적으로
인정받았다"는 취지의 언급이 있는지만 판별하세요(예: "피해자로 인정받았어요", "이미 확정받았어요",
"지원 대상으로 결정됐어요" 등). 그런 언급이 없으면 아직 요건을 판별해야 하는 상태로 보고
false를 반환하세요. 요건 충족 여부 자체는 판단하지 마세요.

## 응답 형식

아래 JSON 형식으로만 응답하세요. 다른 텍스트를 덧붙이지 마세요.

```json
{"recognized": false}
```
